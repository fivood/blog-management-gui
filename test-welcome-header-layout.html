技术细节

HTML结构

<article id="top">
  <section>
    <div>
      <h1>标题</h1>
      <p>文本行1</p>
      <p>文本行2</p>
    </div>
    <div>
      <img src="/images/header.png">
    </div>
  </section>
</article>

CSS样式

#top section {
  display: flex;
  gap: 2em;
  align-items: center;
}

#top section > div:first-child {
  flex: 1;
}

#top section > div:last-child {
  flex: 0 0 auto;
}

#top section img {
  max-width: 300px;
  height: auto;
  border-radius: 0.5em;
}

响应式媒体查询

@media only screen and (max-width: 800px) {
  #top section {
    flex-direction: column-reverse;
    text-align: center;
  }
  
  #top section img {
    max-width: 100%;
  }
}

自定义选项

调整图片大小

/* 在 custom.css 中修改 */
#top section img {
  max-width: 400px; /* 改为你想要的大小 */
}

调整间距

/* 在 custom.css 中修改 */
#top section {
  gap: 3em; /* 改为你想要的间距 */
}

图片在左，文本在右

/* 在 custom.css 中添加 */
#top section {
  flex-direction: row-reverse;
}

修改响应式断点

/* 在 custom.css 中修改 */
@media only screen and (max-width: 600px) {
  /* 改为 600px 或其他值 */
}
