# Application Icons

Place application icons in this directory:

- `icon.ico` - Windows icon (256x256)
- `icon.icns` - macOS icon (512x512@2x)
- `icon.png` - Linux icon (512x512)

Icons should be in the following formats:
- Windows: .ico file with multiple sizes (16, 32, 48, 256)
- macOS: .icns file with multiple sizes
- Linux: PNG files in various sizes (16, 32, 48, 64, 128, 256, 512)
